# Changelog

## 0.3.0 (current)

### Added
- Кнопка "Настройки" в левом верхнем углу главного окна
- Окно настроек (600x400) с заголовком "Настройки"
- Switch "Save mode" (true/false) с цветной индикацией (зелёный/красный)
- Поля ввода путей: Nginx path, Sites source path, Hosts path
- Кнопка "Сохранить" для записи путей в config.json
- `config.json` в корне проекта для хранения конфигурации
- `services/config_manager.py` — сервис для загрузки/сохранения config.json
- Быстрый режим выполнения (saveMode=false): все команды объединяются в одну через pkexec

### Changed
- Кнопка "Настройки" статична, всегда зелёная, не реагирует на saveMode
- Все поля в окне настроек (switch и inputs) прижаты к правому краю
- Border во всех кнопках изменён с #777 на #fff
- Генерация конфига использует saveMode для выбора режима выполнения

### Fixed
- Баг с конфликтом кавычек в быстром режиме (echo >> /etc/hosts)
- Баг с сохранением состояния switch (QPushButton не поддерживает isChecked)
- Права на создаваемые файлы явно проставляются через chmod 644

## 0.2.0 (previous)

### Added
- Валидация имени проекта (латиница, цифры, дефис, 3-63 символа)
- Проверка системных требований (nginx, права sudo, директории)
- Проверка на конфликт с существующими конфигами
- Генерация конфига Nginx с проксированием PHP-FPM
- Автоматическое создание symlink в sites-enabled
- Проверка конфигурации nginx -t
- Перезагрузка nginx
- Добавление записи в /etc/hosts
- Подтверждение каждой команды перед выполнением
- Копирование вывода терминала в буфер обмена
- Выполнение команд через pkexec (графический запрос пароля)
- Модульная архитектура (компоненты/сервисы)

## 0.1.0

### Added

- Графический интерфейс на PyQt6
